Predictive Reconstruction of Fragmented LoRa Telemetry

Broken packets are repaired, missing coordinates are estimated, and inferred movement stays visible without being confused with verified telemetry.

Reconstruction Logic

How prediction works: Uses last 2 points for linear interpolation and velocity-based estimation. Blends with lightweight Kalman-inspired smoothing.

1. Packet Integrity Check

Detects broken JSON (missing headers/footers) or NMEA checksum failures. Flags as fragmented.

2. Historical Buffer Analysis

Analyzes node's previous velocity, heading, and pattern of life from last 10 valid packets.

3. Predictive Estimation

Formula: Position(t) = Position(t-1) + Velocity * Δt, adjusted by heading. Kalman filter smooths noise.

4. Confidence Scoring

Confidence = 0.35 + history_depth * 0.1 - time_gap * 0.01. Ranges 0.15-0.92.
